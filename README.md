# Sprint_6

Данный проект содержит набор автоматизированных UI-тестов для учебного сервиса "Яндекс Самокат". Тесты написаны на
**Python** c применением **Pytest** и **Selenium**.
Проект реализован с использованием паттерна **Page Object Model**:

- locators/ - описание локаторов страниц
- pages/ - описание методов страниц
- tests/ - сами тестовые методы

### Покрытие кейсы:

1. **Проверка выпадающего писка "Вопросы о важном" (test_home_page.py):**</br>
   test_faq_question_shows_correct_answer - проверка выпадающего списка внизу главной страницы.</br>
   Тест параметризирован - проверяем текст ответа для каждого вопроса.
2. **Заказ самоката (test_order_page.py):**</br>
   test_order_creation_from_different_entry_points - позитивный сценарий оформления заказа. </br>
   Проверяется две точки входа (кнопка в хедере и кнопка в нижней части основной страницы).

### Запуск тестов

1. Установите зависимости:</br>
   pip install -r requirements.txt
2. Запустите тесты:</br>
   pytest tests
